# Lorenz Attractor

## Description

The Lorenz Attractor is a chaotic attractor that is defined by the following system of equations:

$$
\frac{dx}{dt} = \sigma * (y - x)
$$

$$
\frac{dy}{dt} = x * (\rho - z) - y
$$

$$
\frac{dz}{dt} = x * y - \beta * z
$$


## Parameters

| Variable | Value |
| -------- | ----- |
| $$ \sigma $$ | 10  |
| $$ \rho $$   | 28  |
| $$ \beta $$  | 8/3 |
